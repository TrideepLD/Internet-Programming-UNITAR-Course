Box Model and Text Flow

(One) Allblock-level HTML5 elements have a virtual box drawn around them, based on what is known as the box model.
(Two) When the browser renders elements using the box model, the content of each element is surrounded by padding, a border and a margin.
(Three) The border-width property may be set to any of the CSS lengths or to the predefined value of thin, medium or thick

(Three) The border-styles available are none, hidden, dotted, dashed, solid, double, groove, ridge, inset and outset.
(Four) The border-color property sets the color used for the border.
(Five) The class attribute allows more than one class to be assigned to an element by seperating each class name from the next with a space
